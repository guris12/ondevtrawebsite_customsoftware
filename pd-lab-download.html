Requirements

  • Docker Desktop (or Docker Engine on Linux) — the launcher pulls openroad/orfs:latest on first run (~5 GB).
  • macOS: 12+ (Monterey or newer), Apple Silicon or Intel. XQuartz is auto-installed on first OpenROAD GUI launch.
  • Linux: x86_64, glibc 2.31+, X server for OpenROAD GUI.
  • Windows: 10/11 64-bit. Note: OpenROAD GUI requires a separate X server (VcXsrv or X410) — coming in a future release.
  • Disk: ~7 GB free for ORFS image + per-run results.
  • Sign-in: a free account (one OTP email) gives Starter access to the Sky130 GCD design.
