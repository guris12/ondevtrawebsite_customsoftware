## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3 (custom dark design system — no Bootstrap), JavaScript
- **Design system:** Syne 800 (headings) + Figtree (body), dark cinematic theme (`--void: #030308`), film grain overlay, violet/cyan gradient palette
- **Forms:** [Formspree](https://formspree.io) — replace `YOUR_FORM_ID` in `contact.html` with your Formspree form ID
- **Deployment:** AWS S3 + CloudFront (static hosting)
- **SEO:** JSON-LD structured data (Organization, SoftwareApplication, Service, ContactPage, FAQPage), Open Graph, Twitter Cards, sitemap.xml, robots.txt
